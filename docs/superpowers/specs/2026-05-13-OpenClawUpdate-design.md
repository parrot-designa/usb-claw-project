# OpenClaw 更新工具 - 设计规格

## 概述

- **项目名称**：OpenClawUpdate
- **类型**：Windows 桌面应用程序（WPF）
- **用途**：更新 OpenClawPro 便携版应用，支持完整包更新或单文件夹更新
- **第一阶段目标**：完成 UI 框架，文件/目录选择功能，按钮预留后期实现

## 界面规格

### 窗口属性
- **尺寸**：450 × 280 像素
- **标题**：OpenClaw 更新工具
- **可调整**：否（固定大小）
- **窗口样式**：标准 Windows 窗口（带最小化、关闭按钮）

### 布局结构
```
┌────────────────────────────────────────┐
│  标题栏（标准窗口）                      │
├────────────────────────────────────────┤
│                                        │
│  当前目录：                             │
│  [____C:\Path\To\Folder____] [浏览]    │
│                                        │
│                                        │
│        [完全重装]    [一键更新]          │
│                                        │
└────────────────────────────────────────┘
```

### 组件说明

| 组件 | 类型 | 说明 |
|------|------|------|
| 路径文本框 | TextBox (ReadOnly) | 显示当前选择的目录路径 |
| 浏览按钮 | Button | 打开目录选择对话框 |
| 完全重装按钮 | Button | 后期实现全量覆盖更新逻辑 |
| 一键更新按钮 | Button | 后期实现增量/选择性更新逻辑 |

### 样式规格
- **字体**：Segoe UI, 12px
- **按钮尺寸**：120 × 36 像素，间距 20px
- **按钮样式**：扁平化，带轻微圆角（3px）
- **颜色方案**：
  - 主色：#189AF4（蓝色）
  - 次色：#0D5EAF（深蓝）
  - 背景：#F5F5F5（浅灰）

## 功能规格

### 第一阶段（当前实现）

1. **自动获取程序目录**
   - 启动时获取 exe 所在目录路径
   - 设置为默认更新目录

2. **目录浏览功能**
   - 点击"浏览"按钮弹出 FolderBrowserDialog
   - 选择新目录后更新文本框内容
   - 限制只能选择目录

3. **按钮预留**
   - 两个按钮显示在界面
   - 点击暂不执行实际操作（预留后期实现）

### 第二阶段（后期实现）

1. **完全重装**
   - 清空目标目录
   - 解压完整更新包到目标目录

2. **一键更新**
   - 比对源和目标文件
   - 只更新有变化的文件夹/文件
